# 🧠 Booster

Aplicación diseñada para **inmobiliarias** que optimiza el proceso de entrevistas con clientes.

Las entrevistas se **graban desde la aplicación** y luego los audios se procesan utilizando **IA para generar transcripciones automáticas**, evitando tener que escribir o escuchar nuevamente la conversación.

### Funcionalidades

• grabación de entrevistas
• envío automático de audio
• transcripción con IA
• registro de información del cliente
• optimización del flujo de trabajo comercial

### Tecnologías

Flutter
Servicios de IA
Backend API
